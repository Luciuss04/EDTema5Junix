🧪 Tema 5 – Diseño y realización de pruebas con JUnit
Proyecto de ejercicios de testing unitario con JUnit 5 para el módulo de Entornos de Desarrollo.

📋 Descripción
Este proyecto contiene la implementación y las pruebas unitarias de los ejercicios del Tema 5.
Para cada ejercicio se analiza el método, se calcula la complejidad ciclomática y se diseñan
los casos de prueba necesarios para cubrir todas las ramas del código.

🛠️ Tecnologías

☕ Java 17
🧪 JUnit 5.10.0
🐘 Gradle 8.x


📁 Ejercicios

Ejercicio 1 – signo → Determina el signo de un número entero
Ejercicio 2 – clasificarEdad → Clasifica una edad en una etapa de la vida
